Naming Convention

Naming attribute can be either of name, class or id.

Name must not start with a number

Name must not have a space in between

Relevent naming is a good way of naming elements

While joining two words either snakecase, camelcase, underscore or hyphen can be used.

W3C => world wide web consortium restricts the use of same id multiple times within a page.
