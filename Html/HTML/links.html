HTTP VS HTTPS

Any website that has server layer security (SSL) is termed as https.

=> Form data are encrypted when transferring from origin to server.

if you try to connect any website without ssl with https, privacy error message will be displayed on the screen. But, if you try to connect any website with https via http only, you'll be redirected to https by default.

So, the safest approach would be to try connecting via http in case you don't know if it has https or not.

WWW VS NON WWW

If any website is programmed to load using www, you can still access it via non www url and the server itself will redirect it to www or vice versa

LINK FARMING

Amount of External Links to any website defines its popularity, trustworthiness and shows the quality of the contents within it are very high.

Links from higher authority (popular) sites to relatively less popular sites will increase the popularity of lesser known sites

Links from lesser known sites to popular sites have no influence on popularity of popular sites.

External links must always be placed cautiously on your web pages.

Always use rel="nofollow" along with your external links.
